### Installation

To install WiFiID, simply:

    $ git clone https://github.com/dwisiswant0/WiFiID

For latest update version:

    $ git pull

### Requirements

WiFiID works with PHP 5.3, 5.4, 5.5, 5.6, 7.0, 7.1, and HHVM.

### Quick Start and Run

	$ php WiFiID.php
